Spy.js

Modern user bahavior analysis tool

Spy.js is modern, open-source tool to analysis users in single page applications.

Why Spy?

Unlike others analysis tool (such as Google Analysis, Inspectlet), you can save the user data in your own data storage.

Spy is lighweight but powerful, can be customized based you needs.

Installation

npm install --save spy.js

How to use?

Run spy.start() to start spy.

Run spy.upload(name, interval) to upload to default firebase cloud.

Run spy.show(name) to display all records.
